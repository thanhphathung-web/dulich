4
Kết nối domain & SSL tự động
Vercel cấp SSL miễn phí qua Let's Encrypt — chỉ cần trỏ DNS đúng
Trỏ DNS — Namecheap / bất kỳ registrar nào
ℹ️
Vào Vercel Dashboard → Project → Settings → Domains
Nhập baggiotravel.com → Vercel sẽ cho bạn DNS records cần thêm vào registrar.
TypeName / HostValueMục đích
A @ 76.76.21.21 Root domain → Vercel IP
CNAME www cname.vercel-dns.com www subdomain → Vercel
CNAME api cname.vercel-dns.com api.baggiotravel.com (optional)
⏱ DNS propagation thường mất 5–60 phút. Kiểm tra tại dnschecker.org
🔒
SSL tự động — không cần làm gì thêm
Sau khi DNS propagate, Vercel tự cấp chứng chỉ SSL từ Let's Encrypt và renew tự động mỗi 90 ngày. URL của bạn sẽ là https://baggiotravel.com với ổ khóa xanh.

.github/workflows/deploy.ymlyaml
name: Deploy Baggio Travel

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

env:
  VERCEL_ORG_ID: ${{ secrets.VERCEL_ORG_ID }}
  VERCEL_PROJECT_ID: ${{ secrets.VERCEL_PROJECT_ID }}

jobs:
  test-and-deploy:
    runs-on: ubuntu-latest
    steps:

      - name: Checkout code
        uses: actions/checkout@v4

      - name: Setup Node.js 20
        uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: 'npm'

      - name: Install dependencies
        run: npm ci

      - name: Lint check
        run: npm run lint

      - name: Security audit
        run: npm audit --audit-level=high

      - name: Install Vercel CLI
        run: npm install -g vercel@latest

      # Pull env vars từ Vercel (dùng cho build)
      - name: Pull Vercel Environment
        run: vercel pull --yes --environment=production --token=${{ secrets.VERCEL_TOKEN }}

      - name: Build Project
        run: vercel build --prod --token=${{ secrets.VERCEL_TOKEN }}

      # Chỉ deploy production khi push vào main (không phải PR)
      - name: Deploy to Production
        if: github.ref == 'refs/heads/main' && github.event_name == 'push'
        run: vercel deploy --prebuilt --prod --token=${{ secrets.VERCEL_TOKEN }}

      # Notify Slack sau khi deploy thành công
      - name: Notify Slack
        if: success() && github.ref == 'refs/heads/main'
        uses: slackapi/slack-github-action@v1.27
        with:
          payload: |
            {"text": "✅ Baggio Travel deployed to production!\nhttps://baggiotravel.com"}
        env:
          SLACK_WEBHOOK_URL: ${{ secrets.SLACK_WEBHOOK_URL }}
⚠️
Thêm GitHub Secrets
Vào GitHub repo → Settings → Secrets → Actions, thêm 4 secrets:
VERCEL_TOKEN (lấy tại vercel.com/account/tokens) · VERCEL_ORG_ID · VERCEL_PROJECT_ID (trong .vercel/project.json sau khi link) · SLACK_WEBHOOK_URL
