Đổi tên file chính thành index.html
File baggio-travel.html (landing page) cần được đổi tên thành index.html để Vercel nhận ra đây là trang chủ. Nhấn vào file → Edit → đổi tên ở thanh địa chỉ.
baggio-travel.html → index.html
💻
Hoặc dùng Git trên terminal (nhanh hơn nếu đã có Git)
Nếu bạn có Git cài sẵn, mở Terminal trong thư mục chứa các file HTML và chạy 4 lệnh bên dưới. Nếu chưa có Git, dùng cách kéo thả ở trên là ổn.
Terminal — chạy trong thư mục chứa các file HTML
# Di chuyển vào thư mục chứa các file HTML $ cd ~/Downloads/baggio-files # Khởi tạo Git repo $ git init Initialized empty Git repository in .git/ # Đổi tên file chính thành index.html $ mv baggio-travel.html index.html # Thêm tất cả file $ git add . $ git commit -m "feat: initial Baggio Travel deploy" [main (root-commit)] feat: initial Baggio Travel deploy 12 files changed, 8266 insertions(+) # Kết nối với GitHub repo vừa tạo (thay USERNAME bằng username của bạn) $ git remote add origin https://github.com/USERNAME/baggio-travel.git $ git push -u origin main ✓ Branch 'main' set up to track remote branch 'main' from 'origin'

Cách còn nhanh hơn — kéo thả trực tiếp vào Vercel (không cần GitHub)
